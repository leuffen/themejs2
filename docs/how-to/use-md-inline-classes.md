# How to Use Inline Classes in Markdown

Blog posts can be made more interesting by adding class directives directly into the post's markdown.

*Since we need to see what our chosen CMS can do, these are subject to change.*

Currently, the following class directives actually do anything:

- `bildbox`
- `info-box`

You can also do a `blockquote`, by simply using standard markdown syntax.

Check [this post](/_posts/2024-06-10-neues-gesetz.md) to see a usage example for each.
